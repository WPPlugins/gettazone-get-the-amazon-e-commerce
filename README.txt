== Description ==
**GettaZone**

This plugin will retrieve some products of Amazon and insert it on every single post you have. It will use tag on the post as the keywords. If tag is not available on a post it will retrieve it from global tag. No coding is required. Everything is run automatically. All You have to do is just insert the Security Credetials that provide by the Amazon Affiliate Program.

Features :

1. Bootstrap ready
2. Easy to Manage, Option Panel inside.
3. Custom CSS Editable. If you want to use your own css.
4. Up to 10 Products. Can modified inside the panel. Default is 3 Products.

Absolutely Free.

ENJOY !!

== Installation ==
**How to Install**

1. Download the plugin.
2. Upload it from Your WordPress Dashboard.
3. Activate it. 
4. Point the pointer to Settings Menu, below the settings menu there will be new option menu called GettaZone. Click it.
5. Input Three Main requirements : 1. Access Key, 2. Secret Key, and 3. Track ID. 
6. You can set the Maximum of products will show. Default is 3 and Max is 10, this is according to Amazon Policy.
7. If you had your own CSS, put it at the CSS Box. 
8. Save it and open your post. 

P.S: Remember, every post must had their own TAG, if it doesn't have one. It will query the global TAG.
